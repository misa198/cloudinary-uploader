# Cloudinary Uploader

## Requirement
* Nodejs

## Installation
```bash
$ yarn
# or
$ npm install
```

## Usage
* Step 1: Create .env at root folder.
* Step 2: Insert into .env CLOUD_NAME, API_KEY, API_SECRET.
* Step 3: Open index.js, set folder's name on cloudinary for variable folder.
* Step 4: Add data to assets folder.
* Step 5: Yarn start
